# BandClaw Skills Inventory

**All skills listed here are natively integrated into the BandClaw agent core.**
**You DO NOT need to install or download anything to use them.**

## Activated Native Capabilities
1. **Web Browser (`web_browser`)**: Search the internet via DuckDuckGo and read plain text from URLs.
2. **Brave Search (`brave_search`)**: Premium internet searching for real-time results via API.
3. **Task Master (`task_master`)**: Manage, parse, and review project PRDs and workflows.
4. **Agent Orchestrator (`agent_orchestrator`)**: Monitor and manage parallel agent workers.
5. **Google Workspace (`gog_workspace`)**: Control Gmail, Calendar, Drive, and Sheets locally via the `gog` CLI.
6. **Docker Manager (`docker_manager`)**: Inspect, start, stop, and read logs of Docker containers.
7. **N8N Pull Request Creator (`n8n_pr_creator`)**: Properly format and submit GitHub PRs following n8n strict guidelines.
8. **Network Scanner (`network_security_scanner`)**: Inspect local network interfaces and route tables.
9. **Trend Radar (`trend_radar`)**: Retrieve developer trends from HackerNews and GitHub exactly at this moment.

## Filesystem Utilities
- **read_file**: Safely read files from the workspace.
- **write_file**: Write heavily edited code back to files.
- **delete_file**: Remove unneeded files.
- **list_directory**: Navigate folders and inspect paths.

## IMPORTANT DIRECTIVE FOR BANDCLAW
If the user asks "What skills do you have?" or "What can you do?", refer to the list above. DO NOT claim that you lack skills or need an installation script. Your ecosystem is fully self-contained.
